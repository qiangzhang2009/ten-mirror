# 镜灯之阵（法藏十镜无尽阵）网页模拟器 - 规格文档

## 1. 项目概述

- **项目名称**: 镜灯之阵（法藏十镜无尽阵）网页模拟器
- **项目类型**: 3D交互式可视化/沉浸式体验
- **核心功能**: 在浏览器中实时模拟唐代法藏大师的十镜无尽阵，体验"法界缘起、重重无尽"的华严哲学意境
- **目标用户**: 对佛教哲学、古代光学、历史体验感兴趣的用户

## 3. 功能规格

### 3.1 核心功能

**3.1.1 十镜阵型搭建**
- 8面水平镜（八方）:
  - 正东: (1.75m, 0, 0)，朝西
  - 正西: (-1.75m, 0, 0)，朝东
  - 正南: (0, 1.75m, 0)，朝北
  - 正北: (0, -1.75m, 0)，朝南
  - 东北: (1.237m, 1.237m, 0)，朝西南
  - 西南: (-1.237m, -1.237m, 0)，朝东北
  - 东南: (1.237m, -1.237m, 0)，朝西北
  - 西北: (-1.237m, 1.237m, 0)，朝东南
- 2面垂直镜（天地）:
  - 上方（天）: (0, 0, 1.75m)，朝下
  - 下方（地）: (0, 0, -1.75m)，朝上

**3.1.2 递归反射系统**
- 使用多重CubeCamera实现实时反射
- 支持5/10/15/20层反射深度
- 反射分辨率自适应（远距离降低）
- 镜面材质: 高反射率(0.95) + 轻微色偏

**3.1.3 光源系统**
- 中心点光源（暖白色，3000K）
- 可调节强度（0-100%）
- 真实光照衰减
- 开启阴影投射

**3.1.4 中心参照物**
- 4种可选模型:
  1. 佛像（程序化生成简约造型）
  2. 莲花（花瓣几何体）
  3. 舍利（球体+光晕）
  4. 几何体（十二面体抽象）
- 模型可360°旋转观察

**3.1.5 交互控制**
- 鼠标左键拖拽: 旋转视角（OrbitControls）
- 鼠标滚轮: 缩放
- 键盘R: 重置视角
- 键盘F: 全屏切换
- 键盘H: 显示/隐藏帮助
- 空格键: 暂停/继续自动旋转

### 3.2 视觉增强

**后处理效果**:
- Bloom泛光（强度可调）
- 色调映射（ACESFilmic）
- 轻微暗角效果
- 抗锯齿（FXAA）

**动画效果**:
- 场景加载: 镜面从远处飞入动画
- 光源调节: 平滑过渡
- UI切换: 淡入淡出

### 3.3 数据处理

- 本地存储用户偏好设置
- 帧率监控（可选显示）

## 5. 技术栈

- **渲染引擎**: Three.js (r158+)

- **反射实现**: CubeCamera + Reflector
